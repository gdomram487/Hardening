Introducción

Vamos a configurar la BIOS de un workstation de la marca Lenovo y vamos a proceder a maximizar la seguridad del sistema modificando algunos parámetros de la BIOS para que no se pueda alterar su arranque.
En este caso se trata del siguiente Workstation:

C20 Workstation (ThinkStation)

Establecer contraseña de administración de la BIOS

Establecemos una contraseña para que se solicite cuando se intente acceder a la configuración de la BIOS. Para establecer esta contraseña nos tendremos que dirigir a :

  • Security
  • Set Passwords
  • Set Administrator Password

También podemos utilizar un lector de huellas para introducir la contraseña de la BIOS

Establecer contraseña de usuario de la BIOS

Vamos a establecer una contraseña de usuario para que se solicite en el proceso de boot de nuestro sistema. Podemos establecer la contraseña de usuario, accediendo a:

  • Security
  • Set Passwords
  • Set User Password

Permiso de arranque desde dispostivos USB

Con esta opción podemos deshabilitar los puertos USB de nuestro sistema. No existe una opción específica para deshabilitar el arranque por tanto tendremos que bloquear los puertos USB en su totalidad. Para configurar esto tenemos que acceder a:

  • Devices
  • USB Setup
  • USB Support

Establecer orden de arranque

Ahora vamos a establecer un orden de arranque lógico para añadir seguridad al sistema. Para ello tendremos que seleccionar siempre en primer lugar el disco duro en el que tenemos instalado nuestro sistema operativo, dejando todos los posibles dispositivos externos en un lugar posterior para evitar que por ejemplo puedan introducir un DVD/CD (Live CD) que pueda corromper nuestro sistema.

Activar Secure Boot en BIOS

En esta BIOS en concreto no contamos con la opción de activar Secure Boot.

Activar TPM en BIOS

El módulo TPM (Trusted Plataform Module) es una tecnología de cifrado de información para usuarios almacenada en un chip. Este chip es capaz de almacenar claves cifradas de datos confidenciales de usuarios del equipo para proteger la información.
Para activar esta opción tenemos que acceder a:

  • TPM State
  • Change TPM State
