Introducción

Vamos a configurar la UEFI de mi ordenador personal de sobremesa para añadirle la máxima seguridad posible al sistema y no se pueda manipular su arranque.
En este caso se trata de la siguiente placa base:

MSI B550-A PRO

Establecer contraseña de administración de la UEFI

Establecemos una contraseña para que se solicite cuando se intente acceder a la configuración de la UEFI. Para establecer esta contraseña nos tendremos que dirigir a :

  • Settings
  • Security
  • Administrator Password

Establecer contraseña de usuario de la UEFI

Vamos a establecer una contraseña de usuario para que se solicite en el proceso de boot de nuestro sistema.
Para establecer la contraseña de usuario tenemos que establecer prinmero la contraseña de administrador obligatoriamente. Una vez asignada la contraseña de administrador procedemos a establecer la contraseña de usuario, accediendo a:

  • Settings
  • Security
  • User Password

Permiso de arranque desde dispostivos USB

Con esta opción podemos denegar el acceso a dispostivos USB 3.0. Para configurar esto tenemos que acceder a:

  • Settings
  • Advanced
  • USB Configuration

Establecer boot order/orden de arranque

Ahora vamos a establecer un orden de arranque lógico para añadir seguridad al sistema. Para ello tendremos que seleccionar siempre en primer lugar el disco duro en el que tenemos instalado nuestro sistema operativo, dejando todos los posibles dispositivos externos en un lugar posterior para evitar que por ejemplo puedan introducir un DVD/CD (Live CD) que pueda corromper nuestro sistema.

Activar Secure Boot en UEFI

Es muy importante que activemos la opción de Secure Boot en nuestra BIOS/UEFI para evitar la ejecución de software no firmado y de casi cualquier sistema operativo que no sea Windows. Para configurar Secure Boot en nuestra UEFI tenemos que acceder a:

  • Settings
  • Advanced
  • Windows OS Configuration
  • Secure Boot
